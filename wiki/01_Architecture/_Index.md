## 数据流向

1. **用户输入** → `main.py` → `CoreEngine.process_input()`
2. **上下文加载** → `WikiManager` → 分层读取 `_Index.md`
3. **LLM 生成** → 响应返回 → 记录到 `runtime/active_session/`
4. **会话结束** → 触发 `ReflectionAgent` → 更新 `wiki/`
